Pivot Table

Excel can use large amounts of data. However, spreadsheet users want to make decisions using summary data along with the ability to drill down to individual transactions. Pivot tables allow the spreadsheet users to do this in Excel.

Excel data is best entered as a table rather than as a crosstab. In the image below, I have entered all the data as a list. The regions or the products do not go across the spreadsheet. Instead, I have repeated the region or product in each row, so the data is normalized. The only summary figures are the totals for the Quantity of units sold and the total Sales Revenue.

If you need to normalize your data, you can use Excel PowerQuery’s Unpivot functionality.

A pivot table will summarize the data so that we can answer useful questions: How many laptops did the salesforce in the North region sell? Which region generated the most sales revenue?

So, for this example select A1 through to F25 with your mouse or shift and the cursor keys and then name the range Sales using the Address Box. Next, to insert a pivot table select a cell where you want Excel to insert your pivot table and then hit the accelerator keys Alt->N->V->T. In the Table/Range box type in Sales and press return.

To summarize sales by Region, drag the Region field from the PivotTable Field List down to Row Labels and the Sales Revenue field down to Values. This simply gives you the total sales revenue by region. To see the total sales revenue for laptops, computers and monitors drag Product Type field from the PivotTable Field List down to Column Labels. This gives you a crosstab as in the image below.

If you wanted to answer, the question “what made up the sales revenue of 325,910 for the East region?” you can double click on the Grand Total cell and Excel will insert a new spreadsheet for you with a list of all the details that it summarized in that cell.

To look at a summary of the number of units sold you can drag Quantity down to Values and Excel will update the pivot table with the total units sold in each region.

If you have more fields to view than can be easily summarized in one pivot table then you can use the Report Filter. You can drag any fields to the Report Filter list and then filter the pivot table.
