Basic Data Validation

Excel is very flexible, easy to use software. You can enter any data, anywhere you want. This is good but can also be a problem because it makes it easy to make data entry mistakes.

In this article, I demonstrate how to setup data validation rules to make your data entry more accurate.

To start, select the range of cells you want to validate your data entry for, by dragging the mouse or using the shift and arrow keys. If you hit the shortcut key Alt for the Excel Ribbon menu and then A for Data and then V for the Data Validation group and then V again, Excel will display the Data Validation dialog box.

For this example click on the Allow drop-down and select List. Then enable In-cell drop-down by ticking the box, this will let you select from the drop-down list box, when you enter data in the validated cells. In the Source box, enter "London, Paris, Hong Kong" as your comma separated list as shown in the image below.

Once you have entered your list of valid data click on the Input Message tab. In the title box type "Cities" and in the message box type "Enter a valid city name."

When you are happy with your input message click on the Error tab. Make sure the Show error alert after invalid data is entered tick box is checked. In the Title box type "Cities" and in the Error message type "Enter a valid city name." Then click on OK to close the Data Validation dialog box and return to your spreadsheet.

Now click on one of the cells you selected for validation. You should see a drop-down list displaying London, Paris and Hong Kong. You can just select the correct city name you want using your mouse. The shortcut key to view the drop-down list is Alt+down arrow cursor, and then you can just select the city you want with the down arrow cursor key.

If you go to a blank cell and type Moscow and hit return, you should get the error message.

To add Moscow, as a valid entry just go back to the Data Validation dialog box with Alt+A+V+V and add Moscow after a comma in the Source box. Select the Apply these changes to all other cells with the same settings check box at the bottom of the dialog box, this will highlight your range and ensure your validation setting applies to all of the cells you originally selected.
